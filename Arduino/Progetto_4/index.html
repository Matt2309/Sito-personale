Funzionamento

1. Buzzer

Il buzzer è un dispositivo piezoelettrico di output sonoro.

2. Pilotare il buzzer con Arduino

Per poter utilizzare un buzzer con Arduino, e quindi, produrre delle melodie, è sufficiente utilizzare la funzione builtin 'tone()' di Arduino, l'istruzione da impiegare è la seguente: 

tone(pin,frequenza,durata)

Nel primo campo andrà indicato il pin al quale è collegato il buzzer, nel secondo la frequenza del suono che vogliamo emettere e nel terzo il tempo per il quale questo suono deve essere riprodotto.
Per comporre la melodia sarà quindi necessario comporre una lista di define con le note e la loro frequenza associata e successivamente un array (lo spartito) di quelle note. Per riprodurre il tutto sarà necessario inglobare il codice sopra dentro ad un ciclo for con grandezza pari all'array di note. 

for ( int i = 0; i < sizeof(melody); i ++ ) {
                        int durata= durataNota*velSuono;
                        tone(pinBuzzer, melody[i], durata);
                        delay(durata*1.3);
                        noTone(pinBuzzer);}

Nel caso di questo codice, viene riprodotto un'array di note melody[i], il delay corrisponde alla pausa tra una nota e l'altra, la quale è stata moltiplicata per 1.3, perciò le pause saranno 1/3 più lunghe della durata delle note.
Il comando noTone() serve a bloccare l'emissione del suono, dopo che una nota è stata riprodotta.



Il codice sottostante contiene anche degli array di pause, i quali vengono trattati esattamente come le note
